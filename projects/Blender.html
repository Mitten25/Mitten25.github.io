Who would you kill for a blender?

Topdown gameshow shoot'em up

Primary Roles: Game Design, Backend Engineer, UI/Front End Engineer, and Sound


Year: Summer 2017


Platforms: MS Windows, MAC OS


Tools and Technologies: Unity, Adobe Illustrator, Maya


Description: My summer 2017 project. Blender, for short, is a game based on timing. Each player is able to shoot one arrow at a time. Once this arrow is in play it must hit another player, or be caught. Using dashes, ricochets, and special pickups players battle to the death! Blender is my current project and, barring some polished art, is very close to publishable!
